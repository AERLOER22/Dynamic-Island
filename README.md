# Dynamic Island для Windows 🏝️

Плавный и красивый Динамический Остров (Dynamic Island) в стиле Apple для Windows! 

## 🚀 Главные фишки
- 🎵 **Умный плеер:** красивая анимация эквалайзера и управление музыкой (Spotify, Яндекс и др.)
- 🔊 **Громкость:** стильная панель при изменении звука
- 🎧 **Bluetooth:** уведомления при подключении наушников
- 🎮 **Игровой режим:** автоматически сворачивается в точку, чтобы не мешать в полноэкранных играх и видео

## 📥 Как установить
Перейдите в раздел **Releases** (справа на странице) и скачайте готовый установщик `.exe`!
